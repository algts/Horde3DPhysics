Bullet Physics Sample
----------------------

	Horde3D Sample Application
	--------------------------
	
	Copyright (C) 2007-2008 Volker Wiendl
	
	
	
This sample uses a simple interface to the bullet physics engine by Erwin Courmans
It shows a way to fuse physics and Horde3D 


Input:

	Use WASD to move and the mouse to look around.
	F1 toggles fullscreen mode.
	F9 toggles FPS display
	ESC quits the application.


Notes on content:

	The model was created using blender.
